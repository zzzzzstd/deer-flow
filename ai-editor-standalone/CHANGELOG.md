# 更新日志

## [1.1.0] - 2025-01-04

### 🎉 新功能
- **Tailwind CSS v4 支持** - 升级到最新的 Tailwind CSS v4，使用 PostCSS 配置
- **表格编辑功能** - 完整的表格创建、编辑和格式化支持
  - 通过 Slash 命令 `/` 插入表格
  - 工具栏中的表格操作按钮
  - 表格行列的添加和删除
  - 表格样式和响应式支持
- **流畅动画效果** - 基于 framer-motion 的界面动画
  - 编辑器容器淡入动画
  - 编辑/预览模式切换动画
  - 工具栏和面板的过渡效果

### 🔧 技术改进
- 添加 `@tiptap/extension-table` 系列扩展
- 添加 `framer-motion` 动画库
- 更新 PostCSS 配置以支持 Tailwind CSS v4
- 移除旧的 Tailwind 配置文件
- 优化编辑器扩展配置

### 📝 文档更新
- 更新 README.md 以反映新功能
- 添加安装脚本 `scripts/install-deps.sh`
- 更新功能特性说明
- 添加表格操作说明

### 🎨 样式改进
- 添加完整的表格样式支持
- 深色模式下的表格样式优化
- 表格选中状态和调整手柄样式
- 响应式表格布局

### 🛠️ 开发体验
- 提供一键安装脚本
- 更清晰的项目结构
- 更好的类型支持
